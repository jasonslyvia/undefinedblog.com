第三方代码控制生命周期

如果是你使用了 react-router 等第三方库来控制你组件的生命周期，你可能不能直观的掌握组件什么时候会被 mount，什么时候会被 unmount。

这个时候你可以使用 react-lifecycle mixin，将这个 mixin 添加到你需要观察的组件中，当任何生命周期方法被调用时，你都能在 console 中看到对应的日志。

当然，这个 mixin 不会输出 render 方法的调用。
