不要在U盘上装Ubuntu

2012-08-26 查看源码

这篇文章记录一下我这两天折腾在移动硬盘上折腾装 Ubuntu 12.04 的经历，标题是我从这次装系统的经历中获得的最大感悟：不要在 U 盘上装 Ubuntu！当然如果你的 U 盘是 USB 3.0 并且你的主板支持，那么另当别论，否则不要再犯跟我一样的错误了。


想起来要装Ubuntu 是在硬盘里偶然扫到了之前给学校服务器上装系统时下载的 Ubuntu Server 镜像，又想起手上有一个 16G C10 的 SD 卡，不用浪费了，所以就想在这个 SD 卡上装个系统。于是，这两天的时间都浪费在这上面了。


总之个中曲折就不多说了，一开始用 SD 卡，后来换 U 盘，最后再换移动硬盘，终于在移动硬盘上配置好了我的 Ubuntu 12.04。比起没有图形界面的 Ubuntu Server，装桌面版的 Ubuntu 简直是轻松多了。不过对于从来没有认真用过 Linux 的我来说，还是有必要记录一些东西的。


再重复申明：不要在 U 盘上装 Ubuntu ！包括 Fedora、OpenSUSE 等各种发行版。
